Trabajos en Phyton

Descripción

Estos trabajos los realicé en la Diplomatura de Especialización en Ciencia de Datos para las Ciencias Sociales y la Gestión Pública 2023-02, en los cursos de: Fundamentos de Programación en Python y Python Intermedio para CCSS y Gestión Pública

Resumen

En el primer curso aprendí a utilizar GitHub, la creación de un website académico, formación de variables, funciones, bucles y condicionales. A la vez el uso de programas simples para resolver problemas en Phyton, uso de bibliotecas y módulos de Phyton para el análisis de datos y computación cientifica; aplicación de conceptos de programación funcional orientada a objetos en Phyton; y uso de las API web y scraping de páginas web.

En el segundo curso reforcé mis conocimientos sobre el uso y creación de funciones. De igual manera, aprendí a manejar paquetes para la visualización de data, creación de Mapas dinámicos y estáticos, scrapear páginas web, crear un portafolio web, entre otros.
